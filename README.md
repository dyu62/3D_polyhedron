# PolyhedronNet: Representation Learning for Polyhedra with Surface-attributed Graph  

This is the **official repository** for our **ICLR 2025 paper**:  
**"PolyhedronNet: Representation Learning for Polyhedra with Surface-attributed Graph"**  

## 🔹 Overview  
PolyhedronNet introduces a novel representation learning framework for **polyhedral geometries**, leveraging **surface-attributed graphs** to effectively encode 3D structures. This method extends polygon-based graph learning techniques to polyhedral domains, enabling applications in **computational geometry, urban modeling, and shape analysis**.  

For **2D multipolygon encoding**, please refer to our related work:  
🔗 **[PolyGNN Repository](https://github.com/dyu62/PolyGNN)**
